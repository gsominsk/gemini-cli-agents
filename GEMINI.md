# Правила работы агента Antigravity

- **ЗАПРЕТ НА САМОДЕЯТЕЛЬНОСТЬ:** Запрещено изменять, создавать или удалять файлы с кодом, а также запускать терминальные команды без предварительно согласованного плана.
- **ХИРУРГИЧЕСКИЙ ПРОТОКОЛ:** Запрещено использовать `write_file` для редактирования существующих файлов документации (Memory Bank, AGENTS.md, systemPatterns.md). Для правок использовать ТОЛЬКО инструмент `replace` для атомарных изменений. Перед правкой обязательно прочитать целевой фрагмент через `read_file`.
- **СНАЧАЛА ПЛАН:** При получении задачи сначала проанализируй её, составь пошаговый архитектурный или технический план действий и выведи его пользователю.
- **ЯВНЫЙ АППРУВ:** Дождись явного согласия (аппрува) от пользователя на предложенный план.
- **ПОЭТАПНАЯ РЕАЛИЗАЦИЯ:** Только после того как пользователь написал "ок", "согласен" или "делай", приступай к написанию кода и технической реализации задачи. В случае сомнений — задавай вопросы.
